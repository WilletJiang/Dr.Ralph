# Early Exploration

This directory is the canonical home for autonomous early research work.

Expected artifacts:
- `plan.md` for the decisive validation plan
- `live-log.md` for the running exploration narrative
- one folder per research item or run group
- `agent-runs/` for per-iteration harness transcripts

For each experiment run, save:
- commands
- configs
- raw logs
- result summaries
- interpretation

The goal is not to accumulate scripts forever. Once the major unknowns are resolved, stop exploring, converge the idea, and hand the package to the user for review.
